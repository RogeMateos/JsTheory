Memoization

Memoization is a technique used in programming to optimize the performance of functions by caching the results of expensive function calls. This helps avoid redundant calculations and improves performance, especially in recursive algorithms.

An optimization technique that stores the results of expensive function calls and returns the cached result when the same inputs occur again.

Example: Simple Memoization Function


    function memoizedAdd() {
      let cache = {};

      return function(num) {
        if (num in cache) {
          console.log('Fetching from cache:', num);
          return cache[num];
        } else {
          console.log('Calculating result:', num);
          let result = num + 10; // Example of an expensive calculation
          cache[num] = result;
          return result;
        }
      };
    }

    const add = memoizedAdd();
    console.log(add(5)); // Calculating result: 15
    console.log(add(5)); // Fetching from cache: 15
    console.log(add(6)); // Calculating result: 16

In this example, the result for the input 5 is stored in the cache. When 5 is passed again, the function returns the cached result instead of recalculating it.
