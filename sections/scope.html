Spread and Rest Operators

Spread Operator

The JavaScript spread operator (...) allows us to quickly copy all or part of an existing array or object into another array or object.

It expands elements of an array, object, or iterable into individual elements and is often used to copy arrays, merge objects, or pass arrays as function arguments.

Example: Spreading Arrays


const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];

const combinedArray = [...arr1, ...arr2]; 
console.log(combinedArray); // Output: [1, 2, 3, 4, 5, 6]

Example: Spreading Objects


const person = { name: 'Alice', age: 25 };
const additionalInfo = { country: 'UK', job: 'Developer' };

const mergedPerson = { ...person, ...additionalInfo };
console.log(mergedPerson);
// Output: { name: 'Alice', age: 25, country: 'UK', job: 'Developer' }

Rest Operator

The rest operator (...) allows you to collect multiple elements into a single array. It's often used in function arguments to handle an indefinite number of parameters or to extract remaining properties from an object.

Example: Using Rest in Function Arguments


function sum(...numbers) {
  return numbers.reduce((acc, curr) => acc + curr, 0);
}

console.log(sum(1, 2, 3, 4)); // Output: 10

Example: Rest in Object Destructuring


const person = { name: 'Alice', age: 25, country: 'UK', job: 'Developer' };

const { name, ...rest } = person;
console.log(name); // Output: Alice
console.log(rest); // Output: { age: 25, country: 'UK', job: 'Developer' }

Difference Between Spread and Rest Operators

The key difference is how they are used:

  • Spread: Expands an array or object into individual elements or properties. Used in function calls, array literals, and object literals.
  • Rest: Collects multiple elements into an array. Used in function parameters or destructuring assignments.

Summary:

The same ... syntax is used for both, but the spread operator is used to split things apart, while the rest operator is used to gather things together.
