Destructuring

Object destructuring is a new way to extract elements from an object or an array.

For arrays, destructuring is used to find elements from an array and store them into variables in a simplified form. Consider the following array

Destructuring is a syntax in JavaScript that allows you to unpack values from arrays or properties from objects into distinct variables. It provides a shorthand way to extract data and simplifies working with complex data structures.

Array Destructuring:


      const numbers = [1, 2, 3, 4];
  
      // Traditional way
      const first = numbers[0];
      const second = numbers[1];
  
      // Destructuring
      const [first, second, , fourth] = numbers;
  
      console.log(first); // Output: 1
      console.log(second); // Output: 2
      console.log(fourth); // Output: 4

Object Destructuring:


      const person = { name: 'Alice', age: 25, country: 'UK' };
  
      // Traditional way
      const name = person.name;
      const age = person.age;
  
      // Destructuring
      const { name, age, country } = person;
  
      console.log(name); // Output: Alice
      console.log(age); // Output: 25
      console.log(country); // Output: UK

Renaming Variables:

You can also rename variables while destructuring:


      const { name: fullName, age: yearsOld } = person;
      
      console.log(fullName); // Output: Alice
      console.log(yearsOld); // Output: 25
