Data Types and Dynamic Typing

Data Types

JavaScript has several data types, which are categorized into primitive and reference types:

  • Primitive Types: string, number, boolean, null, undefined, symbol, bigint.
  • Reference Types: object, array, function.

Primitive types hold their values directly, while reference types hold references to their values in memory.


          let str = 'Hello'; // Primitive type
          let num = 42;      // Primitive type
          let arr = [1, 2, 3]; // Reference type
          let obj = { name: 'Alice' }; // Reference type

Dynamic Typing

JavaScript is a dynamically typed language, meaning that variables can hold values of any type and that type can change during runtime. This is different from statically typed languages, where a variable's type is fixed at compile time.


          let dynamicVar = 'This is a string';
          console.log(typeof dynamicVar); // Output: string

          dynamicVar = 100;
          console.log(typeof dynamicVar); // Output: number

In the example above, the type of dynamicVar changes from string to number.
