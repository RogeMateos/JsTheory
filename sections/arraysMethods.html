Arrays Methods Map, Filter, Find, Reduce, Some, Every, Includes

Map

The map method creates a new array by applying a function to each element of the original array.


const people = [
{
name: "Alice",
age: 28,
occupation: "Engineer"
},
{
name: "Bob",
age: 35,
occupation: "Designer"
},
{
name: "Charlie",
age: 40,
occupation: "Teacher"
}
];

const names = people.map(function(person) {
return person.name;
});

console.log(names);

Filter

The filter method takes an array and removes items that do not meet a specified criterion, likely resulting in an array with a length shorter than the original.

First Example


const overThirties = people.filter(function(person) {
return person.age >= 30;
});

console.log(overThirties);

Second Example


const items = [
  { name: 'Bike', price: 500 },
  { name: 'TV', price: 800 },
  { name: 'Laptop', price: 1200 },
  { name: 'Phone', price: 700 },
  { name: 'Headphones', price: 150 }
];
const affordableItems = items.filter(item => item.price <= 100);

console.log(affordableItems);

Find

The find method returns the first element in the array that satisfies a specified condition. If no elements satisfy the condition, it returns undefined.


const people = [
  { name: "Alice", age: 28, occupation: "Engineer" },
  { name: "Bob", age: 35, occupation: "Designer" },
  { name: "Charlie", age: 40, occupation: "Teacher" }
];
const firstEngineer = people.find(function(person) {
return person.occupation === "Engineer";
});

console.log(firstEngineer);

Reduce

The reduce method is used to reduce an array to a single value, which is typically not an array but could be a number, string, or any other data type.


const totalAge = people.reduce(function(accumulator, person) {
return accumulator + person.age;
}, 0);

console.log(totalAge);

Some

The some method checks if at least one element in the array meets a specified condition. It returns true if any element satisfies the condition, and false otherwise.


const hasDesigner = people.some(function(person) {
return person.occupation === "Designer";
});

console.log(hasDesigner); // Logs: true

Every

The every method checks if all elements in the array meet a specified condition. It returns true if all elements satisfy the condition, and false otherwise.


const allOverTwenty = people.every(function(person) {
return person.age > 20;
});

console.log(allOverTwenty); // Logs: true

Includes

The includes method checks if an array contains a specified element. It returns true if the element is found, and false otherwise.


const occupations = people.map(person => person.occupation);
const hasTeacher = occupations.includes("Teacher");

console.log(hasTeacher); // Logs: true

Arrays Methods Map and ForEach difference

Differences

forEach does not return a new array; it returns undefined. map returns a new array with the results of the transformation.

Chat GPT

ForEach

Used for executing a function on each element of an array without returning a new array.


const numeros = [1, 2, 3, 4, 5];

numeros.forEach(x => {
console.log(x); // Logs: 1, 2, 3, 4, 5
});

const res01 = numeros.forEach(x => x * 5);
console.log(res01); // Logs: undefined

Map

Creates a new array by applying a function to each element of the original array.


const numeros = [1, 2, 3, 4, 5];

const res01 = numeros.map(x => x * 5);
console.log(res01); // Logs: [5, 10, 15, 20, 25]
